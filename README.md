# Ecommerce Store Frontend

A simple frontend-only ecommerce application built with HTML, CSS, and JavaScript.

## Pages

- **Home** (index.html): Landing page with featured products
- **Products** (products.html): Product listing page
- **Product Details** (product-detail.html): Individual product view
- **Cart** (cart.html): Shopping cart
- **Checkout** (checkout.html): Checkout form

## Features

- Responsive design
- Add to cart functionality
- Cart persistence using localStorage
- Basic checkout form

## Running the App

Simply open index.html in a web browser to start browsing the store.

Note: This is a frontend-only demo. No backend or payment processing is implemented.
